Available Commands

  • pause - pause/resume stream of twitter data
  • color - change color of fonts (eg. color white)
  • auto [slow,fast,off]- change speed of auto-scrolling (eg. auto slow)
  • users - get the number of users currently connected
  • clear - clears the terminal screen
  • help - shows this help dialog
  • about - shows the about dialog

About This App

  • Powered by OpenShift
  • Build using Node.js and Socket.io
  • Tracking RSS Feeds for a set of keywords, once matching title found, an email will be sent
